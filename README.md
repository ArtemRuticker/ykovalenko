Используя Npgsql, SqlKata и Roslyn Source Generator реализовать пример Анализатора и использующей его Библиотеки классов. При построении сборки в библиотеке классов должны генерироваться файлы с описанием классов  (для каждого отдельный файл) вида "public class ИмяКласса {}", имена классов должны считываться из таблицы в БД Postgres


1.	Скрипт для создания в СУБД простой БД содержащей одну таблицу с одним текстовым полем.
2.	Скрипт (можно в рамках 1) который заполняет таблицу из п.1 3тестовыми данными (например Class1, Class2, Class3)
3.	Проект с реализацией в коде SourceGenerator или IncrementalGenerator который формирует библиотеку классов и маппингом для ODT nhibernate на основе данных из п.2 (пустышки классов содержащих одно поле long Id)
4.	Проект который используя проект из п.3 как анализатор кода и при сборке формирующий библиотеку досупа к БД на основе nhibernate

Срок до 5 дней.
